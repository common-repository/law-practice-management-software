== Description ==

Law practice management software plugin for wordpress is ideal way to manage complete Lawyer's works.

Major features in Law Practice Management include:
1) Case Management
2) Scheduling and Client Management
3) Legale Document Management
4) Real Time Alerts
5) Easy Billing and Payment
6) Client Communication Tools
7) Report and Advance Analytics

== Installation ==

1)FTP Upload
-> Unzip the " lawyers-management.zip " file.
-> Upload the extracted "Lawyer Management" plugin folder into "/wp-content / plugins/" in your WordPress     installation.
-> To Activate the plugin, Go to Plugins and activate the installed plugin.

2)WordPress Upload
Steps for law practice management software plugin Installation:

->Go to WordPress Admin Plugins → click on Add New → click on Upload.
->Unzip the "lawyers-management.zip " file and Upload the " lawyers-management.zip". → click on Install Now.
->Click on Activate Plugin.
->You can see Law Practice Management Software on the dashboard.
